# 用法

### 第一步：安装marked
```npm install marked --save```
 
### 第二步：安装 highlight.js
```npm install highlight.js --save```

### 第三步：
```npm run dev```


### 说明 ， 语法高亮的css文件我直接拷到本地目录下了
代码显示行号，直接修改的是 github.css文件
